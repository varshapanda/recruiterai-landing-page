# RecruiterAI - Landing Page

## Overview
This repository contains the design and development of the **RecruiterAI Landing Page**, built as part of the Founder Intern assessment at **OneSol AI Labs**.  
The goal of this project is to create a conversion-focused, responsive landing page that clearly explains how RecruiterAI solves recruiting pain points using AI-driven automation.

## Project Scope
The landing page demonstrates:
- Clear articulation of hiring pain points
- How RecruiterAI automates end-to-end recruiting workflows
- Measurable impact and results from AI-powered hiring
- Strong visual hierarchy, motion, and interactivity
- Mobile-first, responsive implementation

## Sections Implemented
1. **Hero Section**  
   Central value proposition with animated problem conversations from founders and hiring managers.

2. **How RecruiterAI Works**  
   Three AI-powered workflow visualizations:
   - Automated candidate screening & interview scheduling  
   - Multi-stage interview process automation  
   - Passive candidate re-engagement system  

3. **Impact & Results**  
   Key hiring metrics showcasing speed, quality, and cost improvements.

4. **Integrations**  
   Auto-scrolling showcase of supported hiring platforms (LinkedIn, Naukri, Indeed, AngelList, Instahyre, Wellfound, and more).

5. **Testimonials**  
   Realistic testimonials from founders and hiring leaders.

6. **FAQ**  
   Common questions about AI screening, integrations, pricing, setup, and data security.

7. **Final Call to Action**  
   Clear conversion-focused CTA to start a free trial or schedule a demo.

## Tech Stack
- **React**
- **Tailwind CSS**
- **Framer Motion** (animations & interactions)
- **Lucide Icons**
- **Vite** (development setup)

## Design and UX Principles
- Mobile-first responsive design (mobile, tablet, desktop)
- Consistent use of the provided color palette
- Component-based architecture
- Smooth motion and micro-interactions for better engagement
- Clean, professional SaaS landing page aesthetic
